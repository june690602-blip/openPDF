# CleanPDF Viewer — Google Play 등록정보

## 앱 이름
CleanPDF Viewer

## 간단한 설명 (80자 이내)
광고 없는 무료 PDF 뷰어 — 카톡으로 받은 계약서·견적서·도면을 빠르고 깔끔하게.

## 자세한 설명 (4000자 이내)
CleanPDF Viewer는 광고 없이, 완전 무료로 PDF를 열어보는 가장 깔끔한 뷰어입니다.

■ 이런 분께 좋아요
• 카카오톡·메일로 받은 계약서, 견적서, 도면을 자주 여는 분
• 광고와 결제 유도에 지친 분
• 빠르고 단순한 PDF 뷰어를 찾는 분

■ 주요 기능
• 빠른 열기 — 카톡 등에서 '열기' 또는 '공유'로 바로 실행됩니다.
• 부드러운 확대·이동 — 손가락으로 자유롭게 확대/축소하고, 두 번 탭으로 빠르게 키웁니다.
• 검색 — 문서 안의 단어를 찾아 하이라이트로 표시합니다.
• 목차·페이지 이동 — 긴 문서도 원하는 위치로 한 번에 이동합니다.
• 썸네일 보기 — 전체 페이지를 한눈에 훑어봅니다.
• 텍스트 선택·복사 — 필요한 내용만 길게 눌러 복사합니다.
• 최근 파일 — 봤던 문서를 홈에서 바로 다시 엽니다.

■ 왜 CleanPDF인가요
• 광고 없음 — 팝업도, 배너도 없습니다.
• 완전 무료 — 결제, 구독, 기능 잠금이 없습니다.
• 개인정보 수집 안 함 — 인터넷 권한조차 요청하지 않습니다. 연 파일은 기기를 벗어나지 않습니다.
• 오픈소스 — 전체 소스 코드를 GitHub에 공개합니다 (AGPL v3).

■ 개인정보 보호
이 앱은 어떠한 개인정보도 수집하지 않으며, 파일을 외부 서버로 전송하지 않습니다.
